Overview of SQL.

1) Does ANSI recognize the data type DATE?
Sol:- An ANSI date literal contains no time portion and must be specified in exactly the following format: DATE 'YYYY-MM-DD'

2) Which subdivision of SQL is used to insert values in tables?
Sol:- The subdivision of SQL used to insert values into tables is called Data Manipulation Language(DML). 
include INSERT,UPDATE and Delete used to manipulate data within existing tables.
